# require-return-type

| Property | Value |
|----------|-------|
| Category | style |
| Fixable | No |
| Recommended | No |
| Deprecated | No |

## Description

Require explicit return type annotations on functions. Explicit return types improve code readability and help catch type errors.

## How to Use

Enable this rule in your configuration:

```json
{
  "rules": {
    "require-return-type": "error"
  }
}
```
