# strict-boolean-expressions

| Property | Value |
|----------|-------|
| Category | style |
| Fixable | No |
| Recommended | No |
| Deprecated | No |

## Description

Enforce explicit boolean comparisons in conditions to avoid bugs with falsy values

## How to Use

Enable this rule in your configuration:

```json
{
  "rules": {
    "strict-boolean-expressions": "error"
  }
}
```
