# no-duplicate-code

| Property | Value |
|----------|-------|
| Category | style |
| Fixable | No |
| Recommended | No |
| Deprecated | No |

## Description

Disallow duplicate code blocks. Duplicated code increases maintenance burden and can indicate missing abstractions.

## How to Use

Enable this rule in your configuration:

```json
{
  "rules": {
    "no-duplicate-code": "error"
  }
}
```
