# prefer-readonly

| Property | Value |
|----------|-------|
| Category | style |
| Fixable | No |
| Recommended | No |
| Deprecated | No |

## Description

Suggest using readonly for arrays and objects that are never modified for better immutability guarantees.

## How to Use

Enable this rule in your configuration:

```json
{
  "rules": {
    "prefer-readonly": "error"
  }
}
```
