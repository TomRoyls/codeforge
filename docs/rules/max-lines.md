## Description

Enforce a maximum number of lines per file

## How to Use

Enable this rule in your configuration:

```json
{
  "rules": {
    "max-lines": "error"
  }
}
```

This rule is auto-fixable. Run `codeforge fix --rules max-lines` to apply fixes.
